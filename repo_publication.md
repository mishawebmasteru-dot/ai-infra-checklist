# Repository publication text

Короткое описание для анонса:

`ai-infra-checklist` — русскоязычный чек-лист по выбору инфраструктуры для n8n, Ollama, Open WebUI, Telegram-ботов, AI-агентов и LLM API.

Что внутри:

- быстрый выбор между VPS, API и GPU;
- минимальные конфигурации под разные сценарии;
- типовые ошибки при запуске AI-инфраструктуры;
- базовый стек для n8n и AI-агентов;
- ссылки на дополнительные материалы по VPS, Ollama, Open WebUI, OpenRouter и GPU-хостингу.

Подходит для тех, кто хочет быстро понять, с чего начать: обычный VPS, внешний API, локальная модель или GPU cloud.
